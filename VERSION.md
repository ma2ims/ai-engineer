# Versioning

Repo ini memakai Semantic Versioning sederhana.

Format:

```text
MAJOR.MINOR.PATCH
```

## Meaning

```text
MAJOR = perubahan besar pada struktur belajar
MINOR = penambahan level/module/project baru
PATCH = perbaikan kecil, typo, cleanup, update docs
```

## Current Version

```text
v0.1.0
```

## Version Roadmap

```text
v0.1.0 = initial learning repo + Level 1 Foundation
v0.2.0 = Level 2 LLM Application
v0.3.0 = Level 3 RAG System
v0.4.0 = Level 4 Agent Workflow
v0.5.0 = Level 5 Production AI System
v1.0.0 = complete AI Engineer learning path
```

## Tagging Example

```bash
git tag -a v0.1.0 -m "Initial AI Engineer learning repo"
git push origin v0.1.0
```
